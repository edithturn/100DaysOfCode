# Functions

**Function**

```python
def my_function():
  print("Hello I am a cat!")
```

**Calling a function**
```python
def my_function():
  print("Hello I am a cat!")

my_function()
```

**Arguments**
```python
def my_function(cat_name)
    print(cat_name + "says Meow")

my_function("Mitsifu")
my_function("Tony")
```
